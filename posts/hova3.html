    CSS

    1、CSS的概念及作用 CSS概念：层叠样式表 CSS作用：精确控制网页中的元素（文字、图片、音视频，超链接等）位置，颜色，大小，布局等。 2、CSS的引用方式 方式1：行内样式:写在HTML的开始标签中，只对当前标签起作用

    说明：style:样式,color:文字颜色 font-size:文字大小，font-family:字体 ; 方式2：内嵌样式：在head标签中，当前页面中都可以用

    方式3：外链样式，写在外部css文件中，当前项目中的所有页面都可以用 先在项目文件夹中创建名为：css的子文件夹 【Ctrl+N】,【Ctrl+S】,命名为：style.css，保存在css文件夹中 在style.css中输入 { color:#249DF2; font-size: 20px; font-family:'微软雅黑' ; } 输入link按回车链接css。 三种引入方式的区别：应用范围不同。 2、CSS修饰文字 1、文字颜色 color:red 颜色名表示法 color:#f00; 十六进制表示法（0-9 a-f）web常用 Color:rgb(255,0,0); rgb表示法 Color:rgba(255,0,0,1);rgba表示法,多设置透明度（0-1） 2、文字字体 font-family: "微软雅黑"; 3、文字大小 font-size: 20px; 单位一般为px像素 4、文字粗细 font-weight: bold; //加粗 font-weight: normal;//去除加粗 5、行高 line-height:1.5; 1.5倍行高，用于段落文字 line-height:20px; 固定行高，用于一行文字 6、首行缩进 text-indent: 2em; 7、文字对齐 text-align: center; center 水平居中 left：居左 right:居右 8、文本修饰 text-decoration:none; 去下划线 text-decoration:underline; 加下划线 text-decoration:line-through; 加删除线 3、CSS选择器 1、基本选择器 1.1 标签选择器： 主要用于重置默认样式（共同的样式） body{ } p{ } h1{} 1.2 类选择器:可以多次使用，且一个标签可以使用多个类。 .txt1{ } .与html中class对应 注意命名规范：不能以中文命名，不能以数字开头。 1.3 ID选择器：一个页面只能使用一次 #wz1{ } #与html中的id对应 2、高级选择器 2.1后代选择器 .box1 p{ } 表示class名为box1中的p标签 div p{} 表示div中的p 2.2 子选择器 .box1>p{ } 表示class名为box1中的子（父子）元素 3、CSS的优先级 !important > 行内样式 > id > 类 >标签 4、CSS美化盒子或图片 1、width:宽 2、height:高 3、margin:0 auto;让盒子水平居中 4、background:背景 background:red url('img/1.jpg') no-repeat 10px 20px; /* 颜色 背景图片路径 是否重复 水平位置 垂直位置 */ 说明：是否重复：repeat:默认的，x与y都重复 repeat-x:x方向重复 repeat-y:y方向重复 no-repeat:不重复 位置： 第一个表示水平（left/center/right）， 第二个表示垂直(top/center/bottom) 也可用 10px 20px 背景图的路径：url(../img/1.jpg)一般用于外部css中。 背景中的各个属性与顺序无关。 经验总结： Img插入图片和backgroud中添加背景图的区别 A、img可以右击另存为图片，background不能 B、Img可撑开盒子，background不能 C、Img中可以添加alt属性，添加关键字，background不能 D、Background可以设置图片重复。 5、Border：边框 border:solid 10px #000;/四周相同/ Border-left:solid 10px #000;/左/ Border-right:solid 10px #000;/右/ Border-top:solid 10px #000;/上/ Border-bottom:solid 10px #000;/下/ 说明：solid:实线 dashed:虚线 dotted：点划线 实战：完成一个任意方向的三角箭头
